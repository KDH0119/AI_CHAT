1. API KEY 설정

1-1. AI STUDIO 사이트 접속
https://aistudio.google.com/api-keys

1-2. 왼쪽 하단 Get API Key 클릭

1-3. API 키 만들기 클릭

1-4. 가져온 프로젝트 선택 - Create project - 이름 아무거나 적고 생성

1-5. 키 이름 지정 아무렇게 하고 생성

1-6. 결제 설정 클릭

1-7. 결제 정보, 카드 정보 등등 입력하고 가입.

1-8 생성된 API KEY를 ai_chat_main 사이트 기입란에 대입.

1-9 완료.

구글이 40만원 쌀먹하게 해줌, 90일 기간 제한 있음.

--------------------------------------------------------------

2. 대화하기

메시지 입력 후 Enter 또는 "전송" 버튼
실시간 스트리밍으로 응답 확인
메시지 위에 마우스를 올려 수정/삭제 가능

3. 대화 관리
저장

"대화 저장" 버튼으로 localStorage 저장
자동으로 .txt 파일도 다운로드

불러오기

사이드바의 "📋 저장된 채팅" 목록에서 선택

요약

"대화 요약" 버튼으로 긴 대화 압축
최근 10턴은 원문 보존
